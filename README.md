🪙 Coinix

O **Coinix** é um sistema moderno e intuitivo de controle de gastos pessoais e planejamento financeiro. Desenvolvido para ajudar usuários a gerenciarem suas finanças de forma simples e eficiente, o projeto permite registrar receitas, categorizar despesas, estabelecer metas de economia e visualizar a saúde financeira por meio de uma interface organizada.

## 🚀 Funcionalidades Principais (Planejadas & Implementadas)

* **📊 Dashboard Financeiro:** Visão geral do saldo atual, total de receitas e total de despesas do mês.
* **💰 Controle de Transações:** Entrada facilitada de novas receitas e despesas com descrição, valor, data e categoria.
* **🏷️ Categorização Inteligente:** Separação de gastos por categorias (Ex: Alimentação, Transporte, Lazer, Educação, Moradia).

---

## 🛠️ Tecnologias Utilizadas

O ecossistema do projeto é baseado em tecnologias robustas para garantir desempenho e uma excelente experiência de usuário:

* Back-End: Java e JSP
* **Front-End: HTML, CSS e JS
* **Armazenamento de Dados:** Banco de dados relacional / Estrutura de persistência local
